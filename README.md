# Nodal

Nodal is a Next.js application that generates brand assets for recruitment agencies. Users can input their company name, industry, vibe, and inspiration to generate a complete brand package, including a logo, color palette, and a one-page website.

## Architecture

For a detailed explanation of the project architecture, please see the [Architecture Document](./docs/architecture.md).

## Getting Started

To get started with the project, you'll need to have Node.js and npm installed. Then, follow these steps:

1.  Clone the repository:
    `git clone https://github.com/Tushar98644/nodal.git .`
2.  Install the dependencies:
    `bun install`
3.  Run the development server:
    `bun run dev`
4.  Open your browser to http://localhost:3000 to see the application.

## Scripts

The following scripts are available in the `package.json` file:

- **`dev`**: Starts the development server.
- **`build`**: Builds the application for production.
- **`start`**: Starts the production server.
- **`lint`**: Lints the codebase using Biome.
- **`format`**: Formats the codebase using Biome.
